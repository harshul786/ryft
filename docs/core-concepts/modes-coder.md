# Coder Mode

Code analysis and generation with Ryft.

## Overview

Coder Mode is Ryft's default mode for software development tasks.

**Capabilities:**
- Code analysis and understanding
- Documentation generation
- Code refactoring suggestions
- Test generation
- Bug detection and debugging
- Code generation from descriptions

## Skills

### analyze

Understand and explain code.

```bash
ryft> Analyze this function
```

### document

Generate documentation.

```bash
ryft> Document this module
```

### refactor

Suggest improvements.

```bash
ryft> How can I improve this code?
```

### generate

Write new code.

```bash
ryft> Generate a function that...
```

### test

Create unit tests.

```bash
ryft> Generate tests for this class
```

## Usage

```bash
# Activate coder mode (default)
/mode coder

# Ask questions
ryft> Analyze this code
ryft> Generate tests
```
