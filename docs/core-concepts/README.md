## Key Concepts

### Modes

Ryft operates in different modes depending on your task. Each mode provides specific tools and capabilities:

- **Coder** - For code analysis, generation, and modification
- **Browser-Surff** - For web browsing and automation
- **Debugger** - For debugging processes

### Features

Ryft includes core features like token budgeting, multi-provider LLM support, file operations, and more.

### Skills

Skills are AI-powered capabilities that Ryft provides. Each skill focuses on a specific task (analyze, document, generate, test, debug, etc.).
