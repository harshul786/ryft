# Debugger Mode

Process debugging and analysis with Ryft.

## Overview

Debugger Mode provides capabilities for advanced debugging and process analysis.

**Capabilities:**
- Process inspection
- Variable examination
- Stack trace analysis
- Breakpoint management
- Memory analysis
- Performance debugging

## Usage

```bash
# Activate debugger mode
/mode debugger

# Ask debugging questions
ryft> Debug this stack trace
ryft> Analyze memory usage
```

## Skills

### debug

Help identify and fix issues.

```bash
ryft> What's wrong with this code?
```

### analyze-trace

Analyze stack traces and error messages.

```bash
ryft> Explain this error
```

## Tools

- Stack trace analysis
- Error investigation  
- Performance profiling
- Memory analysis
