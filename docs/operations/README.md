## Common Tasks

### Task 1: Analyze Project Files

1. Start Ryft in your project
2. Use file tools to read project files
3. Ask Ryft to analyze

See: [File Tools Guide](./file-tools.md)

### Task 2: Analyze Different Project

1. Use `--cwd` flag or auto-detection
2. File tools automatically work in that directory
3. See results for that project

See: [Working Directory Resolution](./running-from-directories.md)

### Task 3: Global Usage

1. Install globally: `npm link`
2. Use `ryft` from any directory
3. Automatically analyzes current directory

See: [Global Installation](./direct-command.md)
