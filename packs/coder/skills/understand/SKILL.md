# Understand

Understand and explain code functionality, execution flow, and component interactions.

## When to Use

Use this skill when you need to:

- Understand how a function or module works
- Trace execution flow through code
- Identify key components and their roles
- Understand class hierarchies and relationships
- Explain complex algorithms
- Map data flow through the system

## How to Use

1. Read the relevant code files
2. Trace the execution path
3. Identify key functions and their responsibilities
4. Explain the logic and flow
5. Provide concrete examples

## Explanation Includes

- **Purpose**: What the code does
- **Inputs & Outputs**: Parameters, returns, side effects
- **Flow**: Step-by-step execution path
- **Key Components**: Important functions, classes, variables
- **Dependencies**: What it depends on
- **Patterns**: Design patterns used

## Example

"Explain how the sentiment analysis module processes text input and generates scores."

## Output

The skill provides:

- Clear, step-by-step explanation
- Code references and line numbers
- Execution flow diagrams (in text)
- Example inputs and outputs
- Related concepts and dependencies
