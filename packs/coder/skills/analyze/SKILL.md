# Analyze

Analyze and research code projects to understand their structure, dependencies, and functionality.

## When to Use

Use this skill when you need to:
- Understand project structure and organization
- Identify modules, classes, and functions
- Research how components work together
- Understand dependencies and APIs
- Generate documentation from code

## How to Use

1. Read key files to understand the project
2. Identify the main modules and their purposes
3. Map out relationships between components
4. Note important APIs and interfaces
5. Identify patterns and best practices

## Example

"Please research the src/ directory and explain how the authentication system works."

## Output

The skill provides:
- Structured overview of project organization
- Key files and their purposes
- Main components and their relationships
- Important APIs and interfaces
- Dependencies and architecture patterns
